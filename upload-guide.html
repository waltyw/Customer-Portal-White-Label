Setup steps after uploading

1

WHM — Change document root

WHM → Domains → portal.bbizzi.co.uk → Edit → Document Root → /home/yourusername/portal.bbizzi.co.uk/public

2

Create .env file on server

In cPanel File Manager, go to portal.bbizzi.co.uk/, click + File, name it .env. Paste in your real values for DB, SMTP, Stripe, and APP_KEY. Copy from .env.example as a template.

3

Import database schema

cPanel → phpMyAdmin → select your database → Import tab → choose schema.sql → Go.

4

Set folder permissions

In File Manager, right-click storage/attachments and storage/logs → Change Permissions → set to 755.

5

Test login

Visit https://portal.bbizzi.co.uk/login
